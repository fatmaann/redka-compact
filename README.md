# Redka Talk

A key-value LSM database inspired by BitCask.

## How to use
1. Clone this repository using ```git clone --recursive```.
2. Make a build directory:
```bash
mkdir build
cd build
```
3. Invoke cmake
```bash
cmake ..
```
4. Build the app
```bash
make
```
5. Run the app 
```bash
./RedkaTalk
```
